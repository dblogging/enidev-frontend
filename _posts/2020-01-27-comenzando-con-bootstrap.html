Sistema de Grilla

Entender el sistema de grilla de bootstrap, te ayudará a que tus sitios web se adapten a todos los dispositivos.

¿Cómo funciona Bootstrap?

Bootstrap está construido por una serie de archivos CSS y Javascript responsables de asignar características específicas a los elementos de la página.
